Basics

Python is a simple programming language. It is very easy to pick up if you're a first timer. You can use it for developing desktop GUI (graphic user interface) applications, websites and web applications.


Useful points to keep in mind:

  • Python has a very straightforward syntax.
  • Python was made to both look good and be easy to read.
  • Some of its syntax comes from the programming language C.
  • If you get stuck with any python syntax, you can look up anything online.

The most basic thing to learn is how to display something on the screen. Here is what you need to do to display "Hello World!":

print ("Hello World!")


To comment your code in python you simply write a hashtag before the comment, like this:

# This is a comment



Variables

Sometimes you need to assign variables so you can use them later on in your code. The most basic variable types in python are integers, floating point numbers, characters and strings. Here is what you need to do to assign something to a variable:

a = 5

b = "hello"

c = false



Operators

Operators are used to manipulate the value of operands. Some of the most commonly used operators include:

  • Addition (+)
  • Subtraction (-)
  • Multiplication (*)
  • Division (/)
  • Modulus (%)

To understand better, here are some examples:

a + b = 5

c - d = 10

c / a = 2


There are comparison operators as well. They are used to compare the values with each other and decide the relation between them. The easiest things to compare are variables but you can compare anything including functions and operations The most commonly used comparison operators include:

  • Equal (==)
  • Not equal (!=)
  • Greater than (>)
  • Less than (<)
  • Greater than or equal (>=)
  • Greater than or equal (<=)

Considering that a = 5 and b = 4, here is an example of variable comparisons:

a == b #This will give you false

a != b #This will give you true

a > b #This will give you true



Lists

Lists can be very useful. They can contain any type of variable and you can have as many as you wish. Here is an example of a list with 3 string items:

fuitList = ["apple", "banana", "strawberry"]


You can access a specific item in the list, but remember that the list starts with index 0. Here an example:

fuitList[1] #This will return "banana"
